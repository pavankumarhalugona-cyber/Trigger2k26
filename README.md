# 🎯 TRIGGER 2K26 - CSE Department Fest

Welcome to the official repository for **Trigger 2K26**, the premier technical fest organized by the **Computer Science & Engineering Department**. This project serves as the central hub for our event's digital presence, featuring a countdown to "The Heist" and an interactive portal to explore our technical challenges.

![Trigger 2K26 Landing Page](./Trigger%20website.png)

## 📂 Project Overview

This project was built to provide an immersive experience for participants of the Trigger 2K26 fest. Inspired by the high-stakes world of digital heists, the website features:

*   **Live Countdown:** A real-time clock ticking down to the launch on May 7th & 8th.
*   **Event Portal:** A "Explore Events" gateway to all our technical and non-technical competitions.
*   **Departmental Branding:** Custom-designed UI reflecting the CSE department's spirit of innovation and precision.

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3 (Custom animations for the "money fall" background effect).
*   **Interactivity:** JavaScript (Logic for the countdown timer and event navigation).
*   **Design:** Inspired by *La Casa De Papel* (Money Heist) aesthetics for a bold, cinematic feel.

## 🚀 How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com
    ```

2.  **Navigate to the directory:**
    ```bash
    cd Trigger2k26
    ```

3.  **Open in your browser:**
    Simply open the `index.html` file to view the landing page in action.
    
4. **Running on the localhost:8000**
